## Mkdocs-material

### How to run mkdocs locally?

- Create your virtual env:

```shell
python3 -m venv venv
```

- Open the new env:

```shell
source venv/bin/activate
```

- Install mkdocs-material:

```shell
pip install mkdocs-material
```

- Start the local server:

```shell
mkdocs serve --watch .
```

Access your documentation at `http://127.0.0.1:8000/`
